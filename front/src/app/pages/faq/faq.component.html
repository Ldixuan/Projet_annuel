F.A.Q

  • Est-ce que l’application peut s’appliquer sur une autre pandémie ?

    • Oui cela peut s’appliquer sur une autre pandémie, néanmoins y’a un format de fichier d’entrée à respecter.

  • Est-ce que les résultats sont donné par l’application sont très fiable ?

    • Les résultats donnent une approximation mais ne sont pas très fiable dans quelques cas, car ça reste des prédictions.

  • Les données récoltées sont extraites à partir d’où ?

    • Plusieurs sources de données sont récoltées, soit via une api en open sources ou des données de santés publique.

  • Est-ce que nos données sont enregistrés dans symptôme checker.

    • Aucune donnée n’est sauvegardée sur vous appart l’identifiant de connexion.

  • Est-ce qu’on n’importe quel utilisateur peut mettre à jour les données ?

    • Non, les données sont mises à jour à partir des administrateurs de l’application.

  • Y’a-t-il des prochaines évolutions de l’application ?

    • Oui, un suivi patient, et des résultats plus pertinents sur de nouveaux pays à intégrer.

  • Quand est ce que les données se mettent à jour ?

    • Les données se mettent à jour chaque 7h du matin pour les prédictions et 1 fois par semaine pour le symptôme checker.

  • Est-ce qu’il y’a une version mobile qu’on peut télécharger.

    • Malheureusement non pour le moment, peut-être dans le futur.
